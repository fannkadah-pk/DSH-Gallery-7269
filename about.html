About DSH Gallery

Introduction

DSH Gallery is a comprehensive media management solution designed to handle all types of files including images, videos, audio files, and documents. Our platform provides a seamless experience for organizing, viewing, and managing your digital assets.

Features

  • Support for all image formats (JPG, PNG, GIF, BMP, WEBP)
  • All video formats supported (MP4, AVI, MOV, MKV)
  • Complete audio file compatibility (MP3, WAV, OGG, FLAC)
  • Document viewer for PDF, DOC, XLS, and more
  • Dark/Light theme support
  • Automatic backup and restore functionality
  • Trash management system
  • WhatsApp integration for instant support
